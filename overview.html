Custom NPCs API

This is the API for Custom NPCs scripting.

Basic knowledges about JavaScript are essential. For complete beginners, check the Mozilla JavaScript tutorial. Some knowledges on Java are recommended as well. Note that Java and JavaScript are two different programming languages.

A player has to be an OP to access scripts.

To start scripting, either:

  • use the Scripter item to interact (default control: mouse right click) with the air, an NPC, or a scripted block; or
  • sneak (default control: left shift) and interact (default control: mouse right click) with an scripted item in creative mode.

Next, in the opened GUI, toggle "Enabled" to "Yes", create a new tab, and start writing your script.

Sometimes, the "Language" button below the "Enabled" button is grayed out. In this case, copy nashorn.jar from the lib/ext folder of your Java installation folder into your Minecraft mods folder and restart Minecraft. If the world is running on a server, this is only required to be done on the server side.

To get started, read the description of the {@link noppes.npcs.api.event} package.

A Discord server is available for discussion on scripting.
